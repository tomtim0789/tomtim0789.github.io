## 写作与笔记

- **本博客**：用 [Astro](https://astro.build) 搭建，Markdown 写作
- 日常笔记：本地的纯文本 + 双链

## 开发

我大部分时间在终端和编辑器里度过。比如新建这个站点，只需要几条命令：

```bash
npm create astro@latest
npm run dev
```

写文章则更简单 —— 在 `src/content/blog/` 下新建一个 `.md` 文件，填好开头的信息就行：

```markdown
---
title: 文章标题
pubDate: 2026-06-13
tags: [标签]
---

正文从这里开始……
```

## 一点心得

不要花太多时间折腾工具本身。工具是为了让你**更快地开始**，而不是成为新的拖延借口。
